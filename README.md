# Hệ Thống Quản Lý Đồ Án

##  Mô tả
Hệ thống quản lý đồ án sinh viên toàn diện

##  Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Firebase Functions
- Database: Firebase Firestore
- Storage: Firebase Storage

##  Cài đặt
```bash
npm install
```

##  Development
```bash
npm run dev
```
